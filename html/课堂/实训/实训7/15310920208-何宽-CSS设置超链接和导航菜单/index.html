新华社北京11月30日电(记者吴晶、史竞男)中国文学艺术界联合会第十次全国代表大会、中国作家协会第九次全国代表大会30日上午在北京人民大会堂开幕。中共中央总书记、国家主席、中央军委主席习近平出席大会并发表重要讲话。他强调，文运同国运相牵，文脉同国脉相连。广大文艺工作者要坚持以人民为中心的创作导向，坚持为人民服务、为社会主义服务，坚持百花齐放、百家争鸣，坚持创造性转化、创新性发展，高擎民族精神火炬，吹响时代前进号角，把艺术理想融入党和人民事业之中，做到胸中有大义、心里有人民、肩头有责任、笔下有乾坤，推出更多反映时代呼声、展现人民奋斗、振奋民族精神、陶冶高尚情操的优秀作品，努力筑就中华民族伟大复兴时代的文艺高峰。
